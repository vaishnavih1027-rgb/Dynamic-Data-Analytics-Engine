## 🚧 Project Status: In Progress

---

## 📌 Overview

The **Dynamic Data Analytics Engine** is an end-to-end analytics system that enables users to upload datasets and automatically generate business insights through **Streamlit and Power BI integration**.

It supports multiple domains including **Retail, SaaS, Fintech, and Enterprise**, dynamically adapting KPIs and dashboards based on user input.

---

## ⚙️ Tech Stack

* Python (Pandas, NumPy, Matplotlib)
* Streamlit
* Power BI
* SQL

---

## 🔄 Project Flow

1. User opens the Streamlit application
2. Selects industry (**Retail / SaaS / Fintech / Enterprise**)
3. Uploads dataset (CSV format)
4. Data cleaning and standardization:

   * Remove null values
   * Normalize column names
   * Validate schema
5. Dataset is transformed into a structured format (`powerbi_dataset.csv`)
6. Power BI dashboard refresh is triggered
7. Industry-specific dashboard loads
8. User navigates using subcategory buttons
9. Visual insights and KPIs are displayed

---

## 🏗️ System Architecture

User
